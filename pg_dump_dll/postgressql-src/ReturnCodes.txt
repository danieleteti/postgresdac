pdmvm_dump:
  0 - OK
  1 - common error (in pg_dump.c)
  2 - error while connecting (wrong username, password, host or something else)
  3 - stdout reassigning error


pdmvm_restore:
  0 - OK
  1 - common error (in pg_restore.c or deeper)
  3 - stdout reassigning error
  4 - stderr reassigning error
 >= 1000 - errors from internal routines
  1000 - could not open input file
  1001 - could not read input file
  1002 - input file is too short
  1003 - input file does not appear to be a valid archive (too short?)
  1004 - input file does not appear to be a valid archive
